# TerraNova Energy

Plataforma de calculo y propuestas para TerraNova Energy.

## Estructura

- `solucion-empresa/`: prototipo editable de la herramienta Solucion Empresa.
- `entregables/`: versiones HTML directas para abrir en navegador o compartir internamente.

## Version actual

La version mas reciente integrada al repositorio incluye:

- calculo de tarifa real desde factura y consumo;
- dimensionamiento estimado en MWp;
- validacion de cliente no regulado;
- minimo comercial TerraNova de 1 MWp;
- beneficios fiscales proyectados;
- ahorro nocturno potencial;
- oportunidad potencial por bonos de carbono;
- logo TerraNova Energy.
