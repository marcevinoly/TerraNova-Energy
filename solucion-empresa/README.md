# Prototipo Solucion Empresas

Abre `index.html` en el navegador para usar la maqueta.

Esta primera version convierte datos tecnicos tipo RETScreen en una lectura financiera para una empresa en Colombia que evalua comprar su energia desde una solucion solar empresarial.

Incluye:

- Datos del cliente, consumo mensual y valor total de factura.
- Tarifa real calculada automaticamente: factura total / kWh consumidos.
- Produccion diaria estimada desde RETScreen o estudio tecnico.
- Inversion en USD con TRM editable para convertir a COP.
- Costo solar entregado en COP/kWh.
- Ahorro anual, payback, VAN, TIR aproximada y flujo de caja.
- Tabla preliminar de beneficios fiscales de Colombia.
- Vista imprimible para guardar como PDF desde el navegador.

Los datos iniciales son de ejemplo. Cambiar cualquier valor recalcula la propuesta automaticamente.

Nota: 1 MWh equivale a 1.000 kWh. El valor de 4.500 kWh/dia se usa como produccion diaria estimada de la planta, no como equivalencia de MWh.
